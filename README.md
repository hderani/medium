
# Medium Articles – Source Code Repository

This repository contains the source code and supporting materials for my articles published on Medium. Each project here is designed to complement an article by providing runnable examples, practical implementations, and additional context.

## About

The goal of this repository is to make the concepts explained in my articles more tangible and reproducible. Instead of just reading about an idea, you can explore working code, experiment with it, and adapt it to your own use cases.

The topics mainly focus on:

* Data Engineering
* Cloud (Azure, Databricks)
* Distributed Data Processing (e.g. Spark)
* Streaming architectures

## Repository Structure

Each article has its own dedicated folder:

```
/article-name-1
  ├── README.md
  ├── notebooks/
  └── ...

/article-name-2
  ├── README.md
  ├── src/
  └── ...
```

Inside each folder you will find:

* A **local README** with article-specific explanations
* The **source code** used in the article
